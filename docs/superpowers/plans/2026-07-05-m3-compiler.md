# M3 â€” Compiler Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Compile a Llama-family `Graph` into a specialized, cached native artifact (`inferno-plan` â†’ `inferno-codegen` â†’ `dlopen`) that produces tokens matching the scalar interpreter within tolerance and runs materially faster than it. *First fast tokens.*

**Architecture:** `inferno-plan` turns `(ModelDesc, Graph, TargetDesc)` into a pure-data `Plan` (conservative fusion islands, an `rs8`-repacked weight image, and a liveness-packed activation arena + KV layout). `inferno-codegen` lowers the `Plan` to a small Loop IR, emits LLVM IR via `inkwell`, writes an object, links it to `model.so`, and pairs it with `weights.bin` + `meta.json`. `inferno-core` hashes inputs, compiles-or-loads the cached artifact, `dlopen`s it, and drives it through a `Backend` trait the generation loop already speaks. Single-threaded; prefill is the decode body wrapped in a token loop over the M2 GEMV kernels.

**Tech Stack:** Rust (edition 2024, workspace), `inkwell` (LLVM 18 bindings), `libloading` (dlopen), `serde`/`serde_json` (sidecar), `insta` (IR snapshots), `proptest`, `cargo-nextest`. LLVM + the linker come from `devenv.nix`.

## Global Constraints

- **Workflows are mise tasks:** `mise run test` (blocking tier), `mise run lint`, `mise run fmt`, `mise run test-full`, `mise run fuzz`, `mise run differential`, `mise run bench-kernels`. CI runs the same names; never hand-roll cargo in docs/CI. Targeted single-test runs during a task may use `cargo test -p <crate>`/`cargo nextest run -p <crate>`; always run `mise run lint` and `mise run test` before a commit.
- **LLVM major version (18) is pinned in `devenv.nix` and MUST match the `inkwell` feature flag** in `inferno-codegen` (`llvm18-0`). All LLVM work happens inside the devenv shell. This is the standing AGENTS.md constraint.
- **`unsafe` is confined** to `inferno-kernels` (exists) plus the new **mmap + `dlopen` + call-function-pointer glue** in `inferno-core`. The workspace denies `unsafe_code` (`Cargo.toml [workspace.lints.rust]`); a crate that needs `unsafe` opts out by declaring its **own** `[lints.rust]` table (not `workspace = true`) with `unsafe_op_in_unsafe_fn = "deny"`, exactly as `inferno-kernels` does. `inferno-plan` and `inferno-codegen` need **no** `unsafe` and keep `[lints] workspace = true`.
- **`ModelDesc` is format-agnostic:** the planner and codegen must never branch on GGUF vs MLX. They see only `ModelDesc`/`Graph`/`DType`/`TargetDesc`.
- **Tensor shapes are row-major, outermost first.** Weight matrices are `[n_out, k]` (`rows = shape[0]`, `k = shape[1]`).
- **Quant formats are dtypes, not ops:** never materialize a dequantized weight tensor in the compiled path. Weights are repacked to the kernel's `rs8` layout at compile time (`KernelSet::pack`) and consumed directly by the GEMV kernels.
- **Kernel ABI is fixed** (from M2, do not change it). Raw GEMV symbol: `unsafe extern "C" fn(y: *mut f32, xq: *const u8, w: *const u8, k: usize, row_start: usize, row_end: usize)` named `inferno_gemv_{f32,q8_0,q4_k}_rs8_{scalar,avx2}` â€” note **`rows` is not an argument**; the kernel walks `row_start..row_end`. Raw activation-quantize symbol: `unsafe extern "C" fn(x: *const f32, y: *mut u8, k: usize)` named `inferno_quantize_row_{q8a,q8k}_{scalar,avx2}` (F32 has none â€” activations are raw f32 LE bytes). Codegen bakes constants and calls these by symbol.
- **Numeric tolerances come only from `inferno-graph/src/tolerance.rs`** (`logits_abs_tol`, `LOGIT_TIE_EPSILON`). Never introduce a new tolerance constant or loosen one to make a red test green.
- **Snapshots (insta):** review with `cargo insta review`; never blind-accept. Keep dumps deterministic.
- **Fixtures under `tests/fixtures/` are generated:** regenerate with `cargo run -p inferno-formats --example gen_fixtures`; never hand-edit.

- [ ] **Step 3: Implement**

Replace the stub with a single forward pass. Rules keyed on `Op`, matching the spec's named boundaries. The Llama builder emits, per layer: `RmsNorm, MatMul(qkv), Rope, Attention, MatMul(o), Add, RmsNorm, MatMul(gate/up), SwiGlu, MatMul(down), Add`. Partition greedily:

- The first `Embed` node â†’ its own `Embed` island.
- An `Attention` node â†’ its own `Attention` island (flush any open island first).
- A run of `{RmsNorm, MatMul, Rope, Add, SwiGlu}` nodes *before* an `Attention` â†’ an `AttnProj` island; a run *after* an `Attention` up to and including the residual `Add` that ends the FFN â†’ an `Ffn` island.
- The final trailing `RmsNorm` + `MatMul` (producing `g.output`) â†’ a `Logits` island.

```rust
use inferno_graph::{Graph, Op};

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum IslandKind { Embed, AttnProj, Attention, Ffn, Logits }

#[derive(Debug, Clone, PartialEq)]
pub struct Island {
    pub kind: IslandKind,
    pub nodes: std::ops::Range<usize>,
    pub label: String,
}

pub fn partition(g: &Graph) -> Vec<Island> {
    let mut out = Vec::new();
    let mut start = 0usize;
    let mut kind = IslandKind::Embed;
    let n = g.nodes.len();

    let flush = |out: &mut Vec<Island>, kind: IslandKind, start: usize, end: usize| {
        if end > start {
            out.push(Island { kind, nodes: start..end, label: format!("{kind:?}").to_lowercase() });
        }
    };

    let mut i = 0;
    while i < n {
        match &g.nodes[i].op {
            Op::Embed { .. } => {
                flush(&mut out, kind, start, i);
                out.push(Island { kind: IslandKind::Embed, nodes: i..i + 1, label: "embed".into() });
                start = i + 1;
                kind = IslandKind::AttnProj; // next island is attention projection
            }
            Op::Attention { .. } => {
                flush(&mut out, kind, start, i); // close the AttnProj run
                out.push(Island { kind: IslandKind::Attention, nodes: i..i + 1, label: format!("attention@{i}") });
                start = i + 1;
                kind = IslandKind::Ffn; // post-attention: o-proj + residual + FFN
            }
            _ => {}
        }
        // Detect the FFN-closing residual Add: an Add whose next node starts a
        // new layer's RmsNorm feeding a qkv MatMul, or end-of-graph before Logits.
        i += 1;
    }
    // The nodes after the last Attention split into Ffn then the trailing Logits.
    // Re-tag the final RmsNorm+MatMul(->output) run as Logits.
    retag_logits(g, &mut out);
    out
}

fn retag_logits(g: &Graph, islands: &mut [Island]) {
    // The output value is produced by the last node; the final RmsNorm+MatMul
    // island is the logits head. Split the last island at the trailing
    // RmsNorm so [.., RmsNorm, MatMul] becomes its own Logits island.
    if let Some(last) = islands.last_mut() {
        // Find the last RmsNorm in the final island's node range.
        if let Some(rms) = last.nodes.clone().rev().find(|&j| matches!(g.nodes[j].op, Op::RmsNorm { .. })) {
            let end = last.nodes.end;
            last.nodes.end = rms; // Ffn island shrinks to before the trailing norm
            // (a helper in the caller pushes the Logits island; see below)
            let _ = (rms, end);
        }
    }
}
```

- [ ] **Step 3: Implement**

Walk the graph's `MatMul` nodes in order; for each, resolve the weight tensor, choose the kernel set for the target ISA (falling back to scalar for correctness parity), pack, and append to the image.

```rust
use inferno_formats::{DType, ModelDesc, read_tensor_bytes};
use inferno_graph::{Graph, Op};
use inferno_kernels::{KernelIsa, kernels_for, reference_kernels};
use inferno_target::TargetDesc;

use crate::{PlanError, Result};

#[derive(Debug, Clone, PartialEq)]
pub struct PackedWeight {
    pub tensor_index: usize,
    pub offset: usize,
    pub len: usize,
    pub rows: usize,
    pub k: usize,
    pub dtype: DType,
    pub isa: KernelIsa,
    pub layout: &'static str,
}

#[derive(Debug, Clone, PartialEq, Default)]
pub struct WeightImageLayout {
    pub image: Vec<u8>,
    pub weights: Vec<PackedWeight>,
}

pub fn build_weight_image(desc: &ModelDesc, graph: &Graph, target: &TargetDesc) -> Result<WeightImageLayout> {
    let mut layout = WeightImageLayout::default();
    for node in &graph.nodes {
        let Op::MatMul { weight, .. } = &node.op else { continue };
        let td = &desc.tensors[weight.0];
        if td.shape.len() != 2 {
            return Err(PlanError::BadWeightRank { name: td.name.clone(), rank: td.shape.len() });
        }
        let rows = td.shape[0] as usize;
        let k = td.shape[1] as usize;
        let ks = kernels_for(&td.dtype, target.isa)
            .or_else(|| reference_kernels(&td.dtype))
            .ok_or_else(|| PlanError::NoKernel(td.dtype.clone()))?;
        let bytes = read_tensor_bytes(desc, td)?;
        let packed = ks.pack(&bytes, rows, k)?;
        let offset = layout.image.len();
        let len = packed.len();
        layout.image.extend_from_slice(&packed);
        layout.weights.push(PackedWeight {
            tensor_index: weight.0,
            offset,
            len,
            rows,
            k,
            dtype: td.dtype.clone(),
            isa: ks.isa,
            layout: ks.layout,
        });
    }
    Ok(layout)
}
```

**Bias note:** `MatMul { bias: Some(_) }` weights (the bias vector) are *not* GEMV-packed â€” bias is applied by generated LLVM after the GEMV (Task 10). Do not add bias tensors to the weight image here; bias stays a plain f32 tensor read at codegen time. Only the quantized/f32 weight matrix is packed.

```rust
/// Value `v` is live from its producer (node v-1) to its last consumer.
fn live_range(graph: &inferno_graph::Graph, v: usize) -> (usize, usize) {
    let producer = v - 1;
    let last_use = graph.nodes.iter().enumerate()
        .filter(|(_, n)| n.inputs.contains(&v))
        .map(|(i, _)| i)
        .max()
        .unwrap_or(producer); // graph.output has no consumer; lives to the end
    (producer, last_use)
}

pub(crate) fn live_overlaps(graph: &inferno_graph::Graph, a: usize, b: usize) -> bool {
    let (a0, a1) = live_range(graph, a);
    let (b0, b1) = live_range(graph, b);
    a0 <= b1 && b0 <= a1
}
```

```rust
use inferno_graph::{Dim, Graph};
use crate::{Result, weights::WeightImageLayout};

#[derive(Debug, Clone, PartialEq)]
pub struct ValueSlot { pub value: usize, pub offset: usize, pub len_elems: usize }

#[derive(Debug, Clone, PartialEq, Default)]
pub struct ArenaLayout {
    pub slots: Vec<ValueSlot>,
    pub total_f32: usize,
    pub act_scratch_off: usize,
    pub act_scratch_bytes: usize,
}

fn value_len(graph: &Graph, v: usize, max_seq_len: usize) -> usize {
    let shape = &graph.nodes[v - 1].out_shape;
    shape.0.iter().map(|d| match d {
        Dim::Const(c) => *c as usize,
        Dim::Seq => max_seq_len,
    }).product()
}

pub fn plan_arena(graph: &Graph, weights: &WeightImageLayout, max_seq_len: usize) -> Result<ArenaLayout> {
    let n = graph.nodes.len();
    // First-fit liveness allocation over f32-element offsets.
    #[derive(Clone)]
    struct Free { off: usize, len: usize }
    let mut slots: Vec<ValueSlot> = Vec::with_capacity(n);
    let mut placed: Vec<ValueSlot> = Vec::new();
    let mut high_water = 0usize;

    // Process values in producer order (1..=n).
    for v in 1..=n {
        let len = value_len(graph, v, max_seq_len);
        // Reclaim any placed slot whose value is dead by the time v is produced.
        let mut frees: Vec<Free> = Vec::new();
        for s in &placed {
            if !live_overlaps(graph, v, s.value) {
                frees.push(Free { off: s.offset, len: s.len_elems });
            }
        }
        // Merge adjacent frees, pick first that fits; else bump the high-water mark.
        frees.sort_by_key(|f| f.off);
        let mut offset = None;
        for f in &frees {
            if f.len >= len { offset = Some(f.off); break; }
        }
        let off = offset.unwrap_or_else(|| { let o = high_water; high_water += len; o });
        let slot = ValueSlot { value: v, offset: off, len_elems: len };
        if off + len > high_water { high_water = off + len; }
        placed.push(slot.clone());
        slots.push(slot);
    }

    // Quantized-activation scratch: worst-case packed activation bytes across
    // all MatMul weights (the codegen writes q8a/q8k here before each GEMV).
    let act_scratch_bytes = weights.weights.iter()
        .map(|w| packed_act_bytes(&w.dtype, w.k))
        .max()
        .unwrap_or(0);
    let act_scratch_off = high_water * 4; // bytes, after the f32 arena
    Ok(ArenaLayout {
        slots,
        total_f32: high_water,
        act_scratch_off,
        act_scratch_bytes,
    })
}

/// Packed activation byte length per dtype (mirrors the kernel `act_len`).
fn packed_act_bytes(dtype: &inferno_formats::DType, k: usize) -> usize {
    use inferno_formats::DType::*;
    match dtype {
        F32 => k * 4,
        Q8_0 => inferno_kernels::act::q8a_len(k),
        Q4_K => inferno_kernels::act::q8k_len(k),
        _ => 0,
    }
}
```

**Note:** `inferno_kernels::act::{q8a_len, q8k_len}` are `pub` (M2, `act.rs`). If not exported at the path shown, use the `KernelSet::act_len` value captured in Task 3 instead â€” store `act_len` on `PackedWeight` in Task 3 and read it here. Keep the simpler path if the functions are public.

```rust
use inferno_graph::Graph;
use crate::{PlanError, Result};

#[derive(Debug, Clone, PartialEq)]
pub struct KvLayout {
    pub n_layers: usize,
    pub kv_dim: usize,
    pub max_seq_len: usize,
    pub bytes_per_layer: usize,
    pub total_bytes: usize,
}

pub fn plan_kv(graph: &Graph, max_seq_len: usize) -> Result<KvLayout> {
    let kv_dim = graph.n_kv_heads.checked_mul(graph.head_dim)
        .ok_or(PlanError::Overflow("kv_dim"))? as usize;
    // F16 KV, k and v: 2 bytes * 2 tensors.
    let bytes_per_layer = kv_dim.checked_mul(max_seq_len).and_then(|x| x.checked_mul(4))
        .ok_or(PlanError::Overflow("kv per-layer"))?;
    let total_bytes = bytes_per_layer.checked_mul(graph.n_layers as usize)
        .ok_or(PlanError::Overflow("kv total"))?;
    Ok(KvLayout { n_layers: graph.n_layers as usize, kv_dim, max_seq_len, bytes_per_layer, total_bytes })
}
```

Translate each island's graph nodes into `Step`s. For a `MatMul` step, look up the matching `PackedWeight` in `plan.weights.weights` (by `tensor_index == weight.0`), emit `Quantize` (skip for F32) then `Gemv` with the kernel symbol `inferno_gemv_{dtype}_rs8_{isa}` derived from the packed weight's `dtype`+`isa`, then `Bias` if the MatMul had a bias.
```rust
use inferno_formats::{DType, ModelDesc, RopeStyle};
use inferno_graph::{Graph, Op};
use inferno_plan::Plan;
use inferno_plan::island::IslandKind;

// ... Step / IslandCode / LoopIr as in Interfaces ...

fn gemv_symbol(dtype: &DType, isa: inferno_kernels::KernelIsa) -> String {
    let d = match dtype { DType::F32 => "f32", DType::Q8_0 => "q8_0", DType::Q4_K => "q4_k", _ => "f32" };
    let i = match isa { inferno_kernels::KernelIsa::Scalar => "scalar", inferno_kernels::KernelIsa::Avx2 => "avx2" };
    format!("inferno_gemv_{d}_rs8_{i}")
}

pub fn build_loopir(plan: &Plan, graph: &Graph, _desc: &ModelDesc) -> LoopIr {
    let find_weight = |tensor_index: usize| plan.weights.weights.iter()
        .position(|w| w.tensor_index == tensor_index)
        .expect("every MatMul weight was packed in Task 3");

    let mut islands = Vec::new();
    for isl in &plan.islands {
        let mut steps = Vec::new();
        for n in isl.nodes.clone() {
            let node = &graph.nodes[n];
            let out = n + 1;
            match &node.op {
                Op::Embed { .. } => steps.push(Step::Embed { table: 0 /* embed weight tensor */, out }),
                Op::MatMul { weight, bias } => {
                    let wi = find_weight(weight.0);
                    let pw = &plan.weights.weights[wi];
                    if pw.dtype != DType::F32 {
                        steps.push(Step::Quantize { dtype: pw.dtype.clone(), src: node.inputs[0], k: pw.k });
                    }
                    steps.push(Step::Gemv { symbol: gemv_symbol(&pw.dtype, pw.isa), weight: wi, out, rows: pw.rows, k: pw.k });
                    if let Some(b) = bias { steps.push(Step::Bias { bias_tensor: b.0, out, rows: pw.rows }); }
                }
                Op::RmsNorm { weight, eps, head_dim } => steps.push(Step::RmsNorm {
                    src: node.inputs[0], weight: weight.0, eps: *eps, out, head_dim: head_dim.map(|d| d as usize),
                }),
                Op::Rope { theta, style, n_heads, head_dim } => steps.push(Step::Rope {
                    src: node.inputs[0], out, n_heads: *n_heads as usize, head_dim: *head_dim as usize, theta: *theta, style: *style,

                Op::SwiGlu => steps.push(Step::SwiGlu { gate: node.inputs[0], up: node.inputs[1], out }),
                Op::Add => steps.push(Step::Add { a: node.inputs[0], b: node.inputs[1], out }),
            }
        }
        islands.push(IslandCode { kind: isl.kind, steps });
    }
    LoopIr { islands }
}
```

```rust
use inkwell::context::Context;
use inkwell::module::{Linkage, Module};
use inkwell::values::FunctionValue;
use crate::Result;

pub struct LlvmModule<'c> {
    ctx: &'c Context,
    module: Module<'c>,
}

impl<'c> LlvmModule<'c> {
    pub fn new(ctx: &'c Context, name: &str) -> Self {
        Self { ctx, module: ctx.create_module(name) }
    }

    pub fn declare_kernels(&self) {
        let i8p = self.ctx.i8_type().ptr_type(AddressSpace::default());
        let f32p = self.ctx.f32_type().ptr_type(AddressSpace::default());
        let usize_t = self.ctx.i64_type();
        let void = self.ctx.void_type();
        // gemv(y, xq, w, k, row_start, row_end)
        let gemv_ty = void.fn_type(&[f32p.into(), i8p.into(), i8p.into(), usize_t.into(), usize_t.into(), usize_t.into()], false);
        for d in ["f32", "q8_0", "q4_k"] {
            for isa in ["scalar", "avx2"] {
                self.module.add_function(&format!("inferno_gemv_{d}_rs8_{isa}"), gemv_ty, Some(Linkage::External));
            }
        }
        // quantize(x, y, k)
        let q_ty = void.fn_type(&[f32p.into(), i8p.into(), usize_t.into()], false);
        for q in ["q8a", "q8k"] {
            for isa in ["scalar", "avx2"] {
                self.module.add_function(&format!("inferno_quantize_row_{q}_{isa}"), q_ty, Some(Linkage::External));
            }
        }
    }

    pub fn define_entry_points(&self) -> (FunctionValue<'c>, FunctionValue<'c>) {
        let i8p = self.ctx.i8_type().ptr_type(AddressSpace::default());
        let f32p = self.ctx.f32_type().ptr_type(AddressSpace::default());
        let u32p = self.ctx.i32_type().ptr_type(AddressSpace::default());
        let usize_t = self.ctx.i64_type();
        let u32_t = self.ctx.i32_type();
        let void = self.ctx.void_type();
        // prefill(tokens, n, pos_off, weights, kv, arena, logits_out)
        let prefill_ty = void.fn_type(&[u32p.into(), usize_t.into(), usize_t.into(), i8p.into(), f32p.into(), f32p.into(), f32p.into()], false);
        let prefill = self.module.add_function("prefill", prefill_ty, None);
        // decode_step(token, pos, weights, kv, arena, logits_out)
        let decode_ty = void.fn_type(&[u32_t.into(), usize_t.into(), i8p.into(), f32p.into(), f32p.into(), f32p.into()], false);
        let decode = self.module.add_function("decode_step", decode_ty, None);
        // Empty bodies for now.
        for f in [prefill, decode] {
            let bb = self.ctx.append_basic_block(f, "entry");
            let builder = self.ctx.create_builder();
            builder.position_at_end(bb);
            builder.build_return(None).unwrap();
        }
        (prefill, decode)
    }

    pub fn verify(&self) -> Result<()> {
        self.module.verify().map_err(|e| crate::CodegenError::Llvm(e.to_string()))
    }
    pub fn print_to_string(&self) -> String {
        self.module.print_to_string().to_string()
    }
}
```

**Lowering reference (each op mirrors `inferno_graph::ops`, which is the oracle â€” read it and reproduce the exact arithmetic and index order):**
- **Embed** (`ops::embed`): for each token, copy `hidden` f32 from `table[token*hidden ..]` into the output slot row. The embed table is a **weight tensor**; it is not GEMV-packed (it's indexed, not multiplied), so read/dequant it to f32 at compile time and embed it as an LLVM global constant, OR pass it via the weights image as raw f32. **Decision:** dequantize the embed table at compile time and emit it as a private global `[N x float]` constant in the module (tiny models; embed table is small relative to weights). Generated code indexes it.
- **RmsNorm** (`ops::rmsnorm`): `y[i] = x[i] * rsqrt(mean(x^2) + eps) * w[i]`; per-head variant divides the row into `head_dim` chunks and normalizes each. `w` is an f32 weight tensor (dequant at compile time â†’ global constant). Reproduce eps placement exactly.
- **Rope** (`ops::rope`): for each (row, head, i<half) rotate the pair per `RopeStyle` using `theta.powf(-2i/head_dim)`, angle = `pos*freq`. `pos = pos_off + row`. Emit calls to LLVM intrinsics `llvm.sin.f32`/`llvm.cos.f32` (or precompute a per-position table â€” but position is runtime-variable, so compute inline). Match `Interleaved` vs `HalfSplit` index math exactly.
- **SwiGlu** (`ops::swiglu`): `out[i] = (g[i] * sigmoid(g[i])) * u[i]` where `sigmoid(g)=1/(1+exp(-g))`. Use `llvm.exp.f32`.
- **Add** (`ops::add`): elementwise `a[i]+b[i]`.

**MatMul lowering (`Step::Quantize` then `Step::Gemv`, `Step::Bias`):**

**Attention lowering (`Step::Attention`, mirrors `inferno_graph::ops::attention` â€” read it):**

Emit both the KV-append and the attention read in the `Attention` lowering, using the `kv` pointer, `Plan.kv` strides, and `layer`.

**KV cache indexing:** k for layer `l`, position `t`, is at `kv + kv_base(l) + t*kv_dim` (f32 in the arena/kv buffer; the spec says F16 physical KV, but M3 may keep KV as f32 for correctness parity and defer F16 packing to M4 â€” **decision: KV is f32 in M3** to match the interpreter's f32 KV exactly and remove a quantization variable from the differential; update `Plan.kv` byte math to `*4` if you take this path, and note the amendment). v is stored in a parallel region after all k, per layer, matching the interpreter's separate `k`/`v` vecs.

```rust
use inkwell::context::Context;
use inkwell::targets::{CodeModel, FileType, InitializationConfig, RelocMode, Target, TargetMachine};
use inkwell::OptimizationLevel;

pub fn compile(desc: &inferno_formats::ModelDesc, graph: &inferno_graph::Graph,
               target: &inferno_target::TargetDesc, max_seq_len: usize, out_dir: &Path) -> Result<Artifact> {

    let ctx = Context::create();

    module.verify()?;

        &TargetMachine::get_host_cpu_name().to_string(),
        &TargetMachine::get_host_cpu_features().to_string(),
        OptimizationLevel::Aggressive, RelocMode::PIC, CodeModel::Default)
        .ok_or_else(|| CodegenError::Emit("no target machine".into()))?;

    std::fs::create_dir_all(out_dir)?;
    let obj = out_dir.join("model.o");
    tm.write_to_file(module.raw_module(), FileType::Object, &obj)
        .map_err(|e| CodegenError::Emit(e.to_string()))?;

    let so = out_dir.join("model.so");
    let cc = std::env::var("CC").unwrap_or_else(|_| "cc".into());
    let status = Command::new(cc).arg("-shared").arg("-o").arg(&so).arg(&obj).status()?;
    if !status.success() { return Err(CodegenError::Link(format!("linker exited {status}"))); }

    std::fs::write(out_dir.join("weights.bin"), &plan.weights.image)?;
    let meta = build_meta(desc, target, &plan);
    std::fs::write(out_dir.join("meta.json"), serde_json::to_vec_pretty(&meta)?)?;
    Ok(Artifact { dir: out_dir.to_path_buf() })
}
```

```rust
use std::path::{Path, PathBuf};
use sha2::{Digest, Sha256};
use inferno_target::TargetDesc;
use crate::Result;

pub fn content_hash(bytes: &[u8]) -> String {
    let mut h = Sha256::new();
    h.update(bytes);
    format!("{:x}", h.finalize())
}

pub fn cache_key(model_path: &Path, target: &TargetDesc, max_seq_len: usize) -> Result<String> {
    let model_bytes = std::fs::read(model_path)?;
    let mut h = Sha256::new();
    h.update(content_hash(&model_bytes).as_bytes());
    // TargetDesc is Debug+deterministic; hash its debug form (isa, features, caches).
    h.update(format!("{target:?}").as_bytes());
    h.update((max_seq_len as u64).to_le_bytes());
    h.update(env!("CARGO_PKG_VERSION").as_bytes());
    Ok(format!("{:x}", h.finalize()))
}

pub fn cache_dir(key: &str) -> PathBuf {
    let base = std::env::var_os("XDG_CACHE_HOME").map(PathBuf::from)
        .or_else(|| std::env::var_os("HOME").map(|h| PathBuf::from(h).join(".cache")))
        .unwrap_or_else(|| PathBuf::from(".cache"));
    base.join("inferno").join(key)
}
```
